## Agentic Codex

This project showcases what the Codex autonomous agent can do for teams that need an end‑to‑end delivery partner. The interface highlights capability lanes, the delivery workflow, and starter prompts that help users request complete builds instead of piecemeal help.

### Stack

- Next.js 15 (App Router, TypeScript)
- CSS modules with custom gradients and responsive layout rules
- Geist Sans + Geist Mono for typography

### Local development

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to interact with the experience.

### Production build

```bash
npm run build
npm start
```

Deployments are optimized for Vercel (`vercel.json` is unnecessary because the default output works out of the box).
